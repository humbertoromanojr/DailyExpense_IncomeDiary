### Install

## Frontend

Install React<br />
`npx create-next-app@latest DailyExpense_IncomeDiary` <br /><br />

Install React Router<br />
`npx create-react-router@latest` <br /><br />

Clone project to repository:<br />
<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" />
git clone `https://humbertoromanojr@bitbucket.org/humbertoromanojr/DailyExpense_IncomeDiary.git`

<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" /> Go to `DailyExpense_IncomeDiary/client` folder <br />

Install node_modules <br />
<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" /> `npm i`

Run application <br />
<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" /> `npx start`

## API - Backend

<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" /> Go to `DailyExpense_IncomeDiary/api` folder <br />

Install node_modules <br />
<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" /> `npm i`

Run application <br />
<img src="https://drive.google.com/uc?export=view&id=16QqYlur8qtl5ao_XyEb4IthGyveQmELo" alt="" width="22" border="0" /> `npx start`

Create a script in the package.json file:<br />
`"api": "json-server ./src/services/server.json --host your-machine-ip --port 3001"`<br />
